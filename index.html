这份提要按卷二最常见的出题逻辑整理：先抓产品保障与除外，再抓核保与保单条款，然后是理赔，最后是客户服务与监管。适合刷题前快速热身，也适合考前复盘。

第一章：一般保险产品

必背框架
  • 四大保障范围：人身保险、财产保险、责任保险、经济权益保险。
  • 财产保险两大承保方式：指明危险、全险。
  • 汽车保险三大车辆类别：私家车、电单车、商用车辆。
  • 汽车保险三层保障：法令保险、第三者责任保险、综合保险。
核心考点
  • 健康保险在卷二通常归入人身保险，不要误判成独立大类。
  • 指明危险要由列明危险导致损失；全险也不是“什么都赔”，而是除外责任以外的意外损失较广。
  • 战争风险和核风险是标准除外责任，属于超级高频点。
  • 汽车险常考法定最低保障额每次事故一亿港元，以及私家车无索偿折扣的折扣回减机制。
  • 标准自负额通常不减保费，自愿性自负额才常用于换取折扣。
高频陷阱
  • “全险”不等于万能险，自然耗损、渐进性损害、机械故障仍可能不保。
  • 第三者财产损害不等于法定最低汽车保险范围，法定重点是第三者死亡或身体受伤。
  • 家居、旅游、盗窃、玻璃、海上货物题常从“保障范围 / 除外责任 / 法定要求 / 适用对象”换着问。
速记口诀

四类保障先分家，财产再分全险和指明；战争核险标准除外，车险最爱考法定额与无索偿折扣。

第二章：核保及保单用词

必背框架
  • 核保两步：选择风险、决定合同条款。
  • 风险分类两类：实质危险、道德危险。
  • 保单文件三组：投保书、暂保单、保险凭证 / 正式保单。
  • 条款识别四项：履行条款、条件、保证、陈述。
核心考点
  • 一般保险核保是持续过程，不是一次性动作；投保、续保、索偿、风险变化时都可能重评。
  • 重要事实的定义一定要会背：凡会影响审慎保险人决定是否承保、按什么条件承保、保费多少的事实。
  • 投保书是重要核保资料来源，不只是临时文件，常会影响合同地位。
  • 正式保单是合同的重要书面证据；暂保单是临时保障；保险凭证在汽车险有特殊法律意义。
  • 取消条款、退费方式、通知方式都要按保单措辞判断。
高频陷阱
  • “陈述”与“保证”不能混。陈述讲事实，保证的违约后果通常更重。
  • 改善风险、降低风险的事实不一定属于典型必须披露的“重要事实”。
  • 不要把保险凭证和正式保单当成同一个东西。
速记口诀

核保先选风险，再定条款；一般保险年年再看；陈述讲事实，保证最严厉。

第三章：理赔

必背框架
  • 有效索偿四要件：有保因、有保单、守条款、不违法。
  • 无效索偿五常见：保单无效、欺诈、除外责任、违反公共政策、违反保证。
  • 结清方式四类：现金、修理、更换、恢复原状。
  • 争议处理两类：仲裁、保险投诉局。
核心考点
  • 拒赔是严重决定，优良做法不是“早拒”，而是审慎处理并给出充分解释。
  • 通融赔付不是合同上必须赔，而是在没有严格法律责任时作出的付款。
  • 不足额保险常和比例分摊一起考，保额没买足，赔款会缩水。
  • 被保险人出险后常见责任：保留证据、保存残骸、披露其他保险、避免擅自认责。
  • 仲裁通常处理金额争议，不处理责任是否成立；对第三者没有约束力。
高频陷阱
  • “通融赔付”不是正式结清方式本身，而是一种在无严格责任下仍付款的做法。
  • 强制性业务中，延迟报案未必当然让保险人拒绝有效第三者索偿。
  • 保险投诉局不是所有案件都收，要看个人保单、金额门槛和程序状态。
速记口诀

有保单、有保因、守条款、不违法；仲裁看金额，不看责任；投诉局看个人单、金额和程序门槛。

第四章：客户服务

必背框架
  • 客户服务三主线：留住客户、维持声誉、符合法规。
  • 监管来源三组：承保商专业守则、保险业条例、持牌保险代理人操守守则。
  • 服务宣言三元素：服务承诺、透明度、客户义务提醒。
核心考点
  • 差服务会失去中介人支持，优质服务有助留住业务；“持续性”通常指留住生意而非流程不断。
  • 服务宣言会写承诺和透明度，但不会保证“永远最便宜”或“最值”。
  • 承保商专业守则由香港保险业联会制定，常考销售惯例、索偿、投诉纠纷、代理管理。
  • 保险业条例偏公司层面，常考授权、股本、偿付能力、适当人选、再保险要求。
  • 代理人操守守则由保险业监管局制定，常考诚实、公平、披露、适合性、利益冲突和客户资产。
高频陷阱
  • 不要把承保商专业守则和代理人操守守则混为一个发布主体。
  • 回佣、贿赂、破坏正常佣金制度通常不是“灵活做法”，而是违规甚至非法。
  • 透明度不只是“信息多”，而是做法公开交代并且符合法律和道德。
速记口诀

服务差丢中介，服务稳留生意；联会看承保商守则，保监局看代理操守。

考前最值得硬背的 8 组要点

第一章先背

四大保障范围；指明危险与全险；战争与核风险标准除外；汽车险三层保障与一亿港元法定额。

第二章先背

重要事实定义；持续核保；投保书 / 暂保单 / 保单 / 保险凭证；陈述、条件、保证的区别。

第三章先背

有效索偿四要件；无效索偿原因；比例分摊；结清方式；仲裁只看金额；投诉局门槛。

第四章先背

承保商守则、保险业条例、保监局代理守则三套框架；服务宣言、透明度、回佣违规。

总骨架：产品保什么与不保什么 -> 核保看什么与保单怎么生效 -> 出险后何时可赔、如何赔 -> 客户服务和监管如何管。
